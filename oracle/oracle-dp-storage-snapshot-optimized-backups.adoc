= 스토리지 스냅샷 최적화된 백업
:allow-uri-read: 


[role="lead"]
Oracle Snapshot Optimized Backups는 사용자가 데이터베이스를 무중단 제거 모드로 전환하지 않고도 스냅샷을 사용하여 복구를 수행할 수 있도록 하는 시점 복구 방식입니다.

사용자는 백업 모드를 사용하지 않고도 스냅샷에서 복구할 수 있지만, RECOVERY AUTOMATIC을 사용한 완전 복구만 지원됩니다. 스냅샷 생성 시간을 지정하는 SNAPSHOT TIME 방식을 사용한 불완전 복구.
